En el dominio y rango de las funciones racionales se peude decir que la mayoria de veces son los numeros reales con exepcion de que Q debe ser diferente de 0 porque si Q=0 la funcion se hace infinita, en las funciones racionales lo que siempre va a estar retringido es el rango.

Ejemplo: encontrar el dominio y rango de la funcion y= 4x-2/3x+5

Dominio Re. excepto x= -5/3

3x + 5 ≠ 0

3x = -5

x = -5/3

Para calcular el rango se debe despejar x de la ecuacion original

Rango

y = 4x-2/(3x + 5)

y(3x + 5) = 4x -2

3xy + 5y = 4x - 2

3xy -4x = -5y -2

x(3y - 4) = -5y -2

x = 5y - 2 / 3y - 4

3y -4 ≠ 0

3y ≠ 4

y ≠ 4/3

Rango. R excepto X= 4/3
